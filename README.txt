Dashboard Setup for TV Display
================================

1. Make sure your Google Sheet is set to:
   - Anyone with the link -> Viewer

2. Your sheet must have:
   - Tab name: Dashboard_Data
   - F1: Total count
   - D2: Last updated timestamp
   - A1:D7: Main data table
   - H2:I10: Chart data

3. To host via GitHub Pages:
   - Create a new repo on GitHub
   - Upload index.html to the repo
   - Commit & push
   - Go to repo Settings -> Pages -> Enable GitHub Pages
   - Open the provided link in your TV browser

4. The dashboard auto-refreshes when your sheet updates.
